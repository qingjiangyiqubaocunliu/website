# Smart Cup App - UI/UX Prototype

> **A visual demonstration of a future smart drinking experience.**
>
> This project serves as a **UI/UX prototype** for my English class presentation. It visualizes the user interface and interaction flow of a "Smart Cup" mobile application.

---

### 🎨 Project Concept

This is not just a static webpage; it is a **mock-up** of a software interface I designed. The goal is to demonstrate how users might interact with a smart device (the "Smart Cup") through a digital interface.

-   **Design Focus:** User Interface (UI) Layout & Interaction Logic
-   **Target Audience:** Users interested in smart home devices and health tracking.
-   **Presentation Context:** English Class - Technology & Innovation Topic

---
